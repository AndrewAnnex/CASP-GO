# CASP-GO
CASP-GO is the UCL optimisations and batch processing wrapper based on the ASP software.



Libraries required:
1. gdal
2. Opencv
3. Eigen3
4. Qt

Before running CASP-GO:
1. Edit param.xml \ 
  ./casp-go-param/input.txt \ 
  ./casp-go-param/base.txt \ 
  ./casp-go-param/ctx.xml \ 
  ./casp-go-param/stereo.ctx \
  set_env.sh
  
2. source set_env.sh


Run:
./CASP-GO param.xml
